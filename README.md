# Value First — A Diagnostic on Structural Failures

A strategic ebook on the five structural failures stalling digital products and the operating model that fixes them.

## Contents

- `index.html` — Main ebook content
- `styles.css` — Styling
- `script.js` — Interactive components

## Overview

The Value First Delivery Framework is a five-phase operating model:

1. **Diagnose** — Map structural failures and value leaks
2. **Align** — Translate strategy into operating bets
3. **Design** — Define decision rights and delivery cadence
4. **Operate** — Run the system and instrument the loops
5. **Scale** — Compound learning and reduce unit cost
